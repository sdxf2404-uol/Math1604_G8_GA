# 🐍 MATH1604 – Group Project 2025/26  
**Analysis of Python Quiz Responses**

---

### 📘 Overview
This repository contains our Group G8 submission for **MATH1604 – Modelling for Big Data (University of Leeds)**.  
The aim is to analyse 64 Python quiz response files to detect whether the correct‑answer sequence follows a deliberate pattern.

The project demonstrates a full data‑science workflow:  
data downloading, text parsing, collation, statistical analysis, and visualisation — all managed collaboratively via GitHub.
